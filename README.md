# Cloud DevOps Journey

Este repositorio document ami experiencia y aprendizaje para desarollar mejores habilidades para los puestos de Cloud Engineer / DevSecOps.

## Tecnologias
- AWS
- Docker
- Kubernetes
- Terraform

## objetivo
Alcanzar mejores estandares en cuanto a deteccion de errores de seguridad. Ademas asegurar el proceso de integracion y modificacionde escalabilidad en los servicios de cloud, incluyendo la arquitectura, despligegue y deteccion de errorres, pero por sobre todo la resolucion de problematicas he integracion de mejoras, tomando en cuenta la resiliencia de las problematicas anteriroes.

# Cloud DevOps Journey

Este repositorio documentami experiencia, aprendizaje para desarollar mejor habilidades para los puestos de Cloud Engineer / DevSecOps.

## Tecnologias
- AWS
- Docker
- Kubernetes
- Terraform

## objetivo
Alcanzar mejores estandares en cuanto a deteccion de errores de seguridad y asegurar el proceso de integracion y modificacion en los servicios de cloud, incluyendo la arquitectura, despligegue y deteccion de errorres, pero por sobre todo la resolucion de problematicas.
